# SA-Python-Group14
Geminiにインストール方法を聞いたので載せてます。よろしくお願いします


【1】Pygameが演習の授業でインストールしたPython3.14だと動かないから、まずはPython3.12をインストールしてほしいです↓
https://pythonlinks.python.jp/ja/

【2】↓のバージョンをダウンロードしてください
3.12.10	2025-04-08	 python-3.12.10-amd64.exe	python-31210

【3】ダウンロードしたファイルを開いてインストール
Add Python 3.12 to PATH　という項目にチェックを入れてください

【4】Pygameのインストールお願いします！
VSコードでターミナルを開いて、
pip install -r requirements.txt
というコマンドを打ってください。Pygameがインストールできるらしいです。
